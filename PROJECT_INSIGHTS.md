# Project Insights

RiverWise does not include telemetry, analytics, tracking pixels, or phone-home behavior.

Project interest is measured through GitHub-native signals only:

- Stars
- Issues
- Confirmed gauge reports
- Release activity
- Discussion and tester feedback

Useful tester signals:

- Which provider was used
- Which gauge or station code was tested
- Whether the visual editor worked
- Whether the card loaded after a hard refresh
- Whether observed data rendered
- Whether forecast data rendered when expected
- Browser/device and Home Assistant version

These notes are for public project planning only. They do not require any runtime tracking in the card.
